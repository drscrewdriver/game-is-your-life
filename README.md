# GameLife — 游戏化生活仪表盘

> 柳比歇夫时间统计 × RPG 升级体系 × AI 智能体 × 精力管理

用数据追踪生活，用游戏化保持动力，用 AI 拆解目标，用精力管理避免 burnout。

## 核心理念

柳比歇夫的"时间统计法"证明：**持续的数据追踪 → 自我认知 → 自律习惯 → 持续改进**。但大多数人难以长期坚持。

GameLife 通过四层机制解决这个痛点：

| 层级 | 机制 | 解决的问题 |
|------|------|-----------|
| 即时反馈 | RPG 经验值/等级/属性 | "做了看不到效果" |
| 持续动力 | 习惯连续打卡/成就解锁 | "坚持不下去" |
| 自我认知 | 精力评估/属性面板/热力图 | "不知道自己什么状态" |
| 效能优化 | AI 目标分解/认知负荷匹配 | "目标太大不知从何下手" |

## 功能模块

### Phase 1 — 核心验证（MVP）
- RPG 式习惯追踪（任务 CRUD、XP 计算、等级系统）
- 每日精力评估（睡眠/专注/疲劳三维度）
- 生活属性面板（身体/心智/社交三大类，颜色编码）
- 基础前端仪表盘

### Phase 2 — AI 赋能
- AI 目标分解引擎（大目标 → 里程碑链 → 可执行子任务）
- 治未病时间管理（艾森豪威尔矩阵游戏化、拖延预警）
- 成长可视化（热力图、经验曲线、成就徽章墙）
- 认知负荷匹配（基于精力状态推荐合适难度的任务）

### Phase 3 — 生态完善
- 社交协作（组队 Boss 战、社会问责机制）
- 深度精力管理（睡眠追踪、最佳休息时机）
- 多主题切换、Docker 自托管部署

## 技术栈

